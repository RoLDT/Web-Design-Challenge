Challenge Summary

Latitude vs. Temperature, Cloudiness, Windspeed or Humidity

In this challenge, I'll be demonstrating some findings, through a webpage, from a previous assignment: a Python script was written to find the weather and other characteristics of over 500 cities that are located in different parts of the world.

 

From each city, the Latitude, Longitude, Temperature, Cloudiness, Windspeed and Humidity data was extracted. You can observe that data if you click on the Data button on the Navigation Bar of this webpage. Several visualizations were created through Python to observe connections between these characteristics; specifically using a city's Latitude as a variable.

 

Further, you can also explore the Comparison button, the Plot dropdown menu or the Visualizations Sidebar to find out more about the visualizations that were made.
